# SAÜ-Vision 🎓

> Smart Campus Facility & Activity Management Platform — Sakarya University

## Repositories

| Folder | Description |
|--------|-------------|
| [`sau-vision-backend`](./sau-vision-backend) | Node.js + Express + TypeScript API |
| [`sau-vision-frontend`](./sau-vision-frontend) | React + Vite + TypeScript UI |

## Quick Start

```bash
# Backend
cd sau-vision-backend
npm install
cp .env.example .env
npm run dev        # → http://localhost:4000

# Frontend (new terminal)
cd sau-vision-frontend
npm install
cp .env.example .env
npm run dev        # → http://localhost:3000
```
